# Free fall 

This is a simple example project of a free falling object. 
The full description of the project can be found at http://physics.ammarica.com. 
Scroll down till you get to labs, and click on Free fall.  

![Project Screenshot](showme1.png)

## Description
What is in this folder are:
1 - The code the is used to extract frames from the mpeg4 to single frames. <br>
2 - Additional code if you want to blur part of these images. For example if you do not to show a person dropping the ball. <br>
3 - And excel sheet student can use to populate their data and get a graph for distance vs the t<sup>2</sup>.  <br>
