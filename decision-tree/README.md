# Decision tree

This is a simple tutorial/review of decision trees, what they are used for and how to use them.   
You can find similar projects on https://financial.ammarica.com/.  

## Description
What is in this folder:
1 - PDF file that has a brief and solid review of decision tree.  <br>
2 - Full python code that does the analysis and produces all the graphs. <br>
3 - The data set that I used to illustrate the ideas in the latex file. Just make sure that 
you put the data set file and the python file in the same folder before you run the python code.<br>
